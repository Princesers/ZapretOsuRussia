# Эксперемент
Это ветка тест, для того чтобы предложить достойную альтарнативу Zapret.
# Для работы
Для работы необходим режим администратора.
Не рекомендую использовать программы прокси и bypass во время работы программы, программа сама хорошо работает и обходит практически разные ограничения.
# Если не работает
В этом случае если программа принимает пакеты а не может отсортировать или не может прислать (сами пакеты)  или пересгенерировать варп или подождать примерно 5-10 минут

# Ошибки
если были ошибки с серверами или неисправности с серверами Cloudflare то большее колчество warp не будут работать ведь они полностью взаимодействую с серверами.
Прошлые тесты доказали если сам провайдер блокирует Cloudflare то возможно при определённом сгенерированом warp у вас всё будит работать

# автозагрузка

после включение программы, когда вы подключились она будит работать в фоновом режиме, если даже вы выключите компьютер с подключением она продождит работать.
Для того что бы она не запускалась в автозагрузку нужно отключить сам сгенерированый warp, и закрыть программу, после действий она не будит запускатся со стартом пк.
